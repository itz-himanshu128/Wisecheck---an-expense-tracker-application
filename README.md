# 🪙 WiseCheck - Advanced Expense Tracker

WiseCheck is a premium, high-performance financial management application built with **Next.js 15**, **Supabase**, and **Tailwind CSS**. It provides users with a sleek, intuitive interface to track expenses, manage budgets, and visualize financial trends through interactive analytics.

![WiseCheck Dashboard](public/dashboard-preview.png) *(Placeholder for dashboard image)*

## ✨ Features

- **📊 Dynamic Analytics**: Real-time visualization of spending patterns using interactive charts.
- **💰 Budget Management**: Set and track monthly budgets with visual progress indicators.
- **🗓️ Expense Tracking**: Easily log and categorize daily transactions.
- **🤝 Borrowed & Lended**: Dedicated modules to track money you owe or are owed.
- **🌗 Dark Mode**: Premium UI with seamless dark/light mode transitions.
- **🔐 Secure Auth**: Robust authentication powered by Supabase.
- **📱 Responsive Design**: Fully optimized for desktop, tablet, and mobile devices.

## 🚀 Tech Stack

- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **Database & Auth**: [Supabase](https://supabase.com/)
- **State Management**: [Zustand](https://zustand-demo.pmnd.rs/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Charts**: [Recharts](https://recharts.org/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)

## 🛠️ Getting Started

### Prerequisites

- Node.js 18.17 or later
- A Supabase project

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/itz-himanshu128/Wisecheck---an-expense-tracker-application.git
   cd Wisecheck---an-expense-tracker-application
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up Environment Variables:**
   Create a `.env.local` file in the root directory and add your Supabase credentials:
   ```env
   NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) to see the app in action.
